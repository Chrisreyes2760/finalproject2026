# Real Estate Agency Portal

## Description
This project is a web-based real estate portal developed using PHP and MySQL. The system allows users to browse properties, register accounts, log in, submit inquiries, and manage property listings based on user roles.

---

## Features
- User registration
- User login and logout
- Session management
- Property listings
- Property details page
- Add property functionality for agents
- Inquiry submission
- Role-based access control

---

## Technologies Used
- PHP
- MySQL
- phpMyAdmin
- HTML/CSS
- XAMPP

---

## Database Tables
- users
- properties
- inquiries
- transactions
- favorites

---

## Setup Instructions

1. Install XAMPP
2. Start Apache and MySQL
3. Place the project folder inside:
   C:\xampp\htdocs\
4. Import the SQL file into phpMyAdmin
5. Open the browser and go to:
   http://localhost/finalproject
